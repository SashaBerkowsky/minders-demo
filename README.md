# Minders Feedback Widget

A full-stack feedback collection system with a browser SDK widget and Node.js API.

## Components
- **[SDK](./sdk/)**: Browser widget for collecting user feedback.
- **[Backend](./backend/)**: API server for storing and retrieving feedback.

## Quick Start
1. Clone the repo.
2. Follow setup in [SDK](./sdk/) and [Backend](./backend/) READMEs.
3. Open `sdk/demo.html` to test end-to-end.
